Help & Tips

Fill in the grid so that each row, column and 3x3 square contain one instance of the digits 1-9. There is a single solution, where each digit appears in the grid exactly nine times.

  • Enable the Pencil In toggle to help track candidate digits
  • Click to change game settings
  • Click for more actions:
    • Check Cell -- check if the cell contains the correct digit
    • Check All -- perform the Check Cell action for all cells
    • Reveal Cell -- reveal the solution for one cell
    • Reveal All -- reveal the entire solution (ends game)
    • Reset -- clear all cells and reset the game timer
  • Full keyboard support is offered:
    • Arrow Keys -- move focus between cells
    • Numpad (with Num Lock disabled) or Number Row -- enter digits in focused cell
    • Shift -- pencil in
    • Backspace or Delete -- clear focused cell
