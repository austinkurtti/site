Objective

Fill in the grid so that each row, column, and 3x3 square contain one instance of the numbers 1-9. There is a single solution, where each number appears in the grid exactly nine times.

Tips
  • Enable the Pencil in toggle to track candidate numbers
  • Click to change game settings
  • Click for more actions:
    • Check cell checks if the active cell has the correct number
    • Check all checks if all cells have the correct numbers
    • Reveal cell reveals the correct number for the active cell
    • Reveal all reveals the entire solution (and ends the game)
    • Reset clears all cells and restarts the game timer
    • Share creates a link to a copy of this Sudoku so you can share with friends!
  • Keyboard support:
    • ←↑↓→ to change the active cell
    • 0-9 enter numbers
    • Shift + 0-9 enter penciled in numbers
    • Backspace or Delete clear the active cell
    • Space pause/resume game
