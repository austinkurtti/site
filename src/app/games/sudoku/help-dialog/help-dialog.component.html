Help

Objective

Fill in the grid so that each row, column, and 3x3 square contain one instance of the numbers 1-9. There is a single solution, where each number appears in the grid exactly nine times.

Tips
  • Use the Pencil in toggle to help keep track of possible numbers
  • Click to change game settings, which will persist between games
  • Click for more actions:
    • Hint suggests a cell that may be worth a closer look
    • Check cell checks if the focused cell has correct number
    • Check all checks if all cells have correct numbers
    • Reveal cell reveals the correct number for the focused cell
    • Reveal all reveals entire solution (and ends game)
    • Reset clears all cells and restarts timer
    • Share creates a link to a copy of this Sudoku so you can share with friends!
  • Keyboard support:
    Key(s) Action
    ←↑↓→ Change the focused cell
    0-9 Enter number
    Shift + 0-9
    Enter penciled in number
    This browser does not support using Shift + Numpad with Num Lock ON
    Backspace or Delete Clear cell
    Space Pause/resume game
    Alt + H Hint
    Alt + C Check cell
    Alt + R Reveal cell
    Alt + S Share
